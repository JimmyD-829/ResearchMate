# ResearchMate 智能投研助手 - 功能全景图
> **个人投资者的财报解读第一站**——用最轻量的方式，解决个人投资者读不懂财报的核心痛点

---

## 一、产品定位与差异化价值

### 定位（与竞品差异）
| 维度 | 竞品现状 | 我们的机会 |
|------|---------|-----------|
| **目标用户** | 多为机构投资者 | 个人投资者/初级投研人员 |
| **功能复杂度** | 全链路功能复杂 | 聚焦"财报解读"单点突破 |
| **AI形态** | 多为通用对话 | 封装为"财报解析Skill" |
| **交互方式** | App为主 | Web端轻量化 + 未来微信接入 |
| **数据来源** | 自有数据库 | 用户上传PDF + 免费API |

### 核心价值
让个人投资者在 **10分钟内**，通过"一句话查财报"的方式完成对一家公司的基本分析，并以结构化、可视化的方式展示关键指标。

## 三、UI/UX设计规范

### 视觉设计原则
1. **便当网格2.0布局**：仪表盘使用卡片式模块，清晰展示关键指标
2. **深色模式优先**：投研产品用户偏好深色背景，减少视觉疲劳
3. **自然语言输入为主**：搜索框+快捷按钮，降低学习成本
4. **进度可视化**：财报分析过程显示进度条，增强掌控感
5. **结果结构化展示**：关键指标用卡片/表格，而非纯文本

### 合规设计要点
1. **AI生成标识**：所有分析结果标注"AI生成，仅供参考"
2. **风险提示**：明确"不构成投资建议"
3. **不存储敏感数据**：PDF分析后不保存原始文件
4. **简单用户确认**：首次使用展示服务条款

### F1 财报智能解读（核心 - 重点优化）

**功能描述**
用户上传财报PDF或通过自然语言查询，系统自动解析并返回结构化数据，展示"财报解读Skill"的标准分析结果。

**核心功能**
| 功能点 | 描述 | 用户价值 | 版本 |
| --- | --- | --- | --- |
| PDF上传 | 支持本地PDF文件上传（≤50MB） | 便捷导入财报文档 | v1.0 |
| 智能解析 | 自动识别公司名称、股票代码、报告期等 | 省去手动翻找 | v1.0 |
| 指标提取 | 营收、净利润、经营现金流、负债率、毛利率 | 快速掌握财务状况 | v1.0 |
| AI摘要 | 200字中文智能摘要 | 快速了解财报重点 | v1.0 |
| 进度可视化 | 分析过程显示进度条 | 增强掌控感 | v1.0 |
| 便当网格布局 | 卡片式展示关键指标 | 结构化，清晰易懂 | v1.0 |
| AI生成标识 | "AI生成，仅供参考"标注 | 合规 | v1.0 |
| PDF清理 | 分析后不保存原始文件 | 隐私保护 | v1.0 |
| 一句话查财报 | 自然语言查询解析 | 降低学习成本 | v1.1 |
| 财报解读Skill | 专业方法论封装 | 标准化分析 | v1.1 |

**用户流程（v1.0）**
```
用户登录 → 点击上传财报 → 选择PDF文件 → 查看解析进度条 → 便当网格展示结果
```

**异常处理**
- 扫描件PDF：提示"该文件为扫描件，OCR功能将在后续版本支持"
- 解析超时60s：提示超时，支持重试
- 未识别字段：显示"未识别"占位，支持手动填写

### F2 市场新闻聚合

**功能描述**
整合多平台财经资讯，提供统一阅读入口。

**核心功能**
| 功能点 | 描述 | 用户价值 |
| --- | --- | --- |
| 关注列表 | 添加/删除关注公司（支持名称/股票代码） | 个性化资讯定制 |
| 资讯聚合 | 按关注公司/行业展示最新资讯 | 一站式获取信息 |
| 智能筛选 | 按公司/行业筛选资讯 | 精准获取所需内容 |
| 每日推送 | 08:00自动推送新闻小结（可关闭） | 及时掌握市场动态 |

**数据来源**
1. NewsAPI（免费层）
2. 东方财富数据API
3. 后续可扩展RSS订阅

---

### F3 情绪指标分析

**功能描述**
基于新闻标题对每条资讯做情感分类，并在公司/行业维度聚合为情绪分数。

**核心功能**
| 功能点 | 描述 | 用户价值 |
| --- | --- | --- |
| 情感分类 | 单条资讯：正面/中性/负面 | 快速判断资讯倾向 |
| 情绪分数 | 公司维度情感加权平均（-100到+100） | 量化市场情绪 |
| 趋势图表 | 近30天情绪趋势图 | 直观展示情绪变化 |
| 情绪旗帜 | 🟢积极 / ⚪️中性 / 🔴消极 | 快速识别当日情绪 |

**算法方案**
使用SnowNLP或VADER中文适配版进行情感分析，无需GPU。
