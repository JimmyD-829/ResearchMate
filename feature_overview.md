# ResearchMate 智能投研助手 - 功能全景图

---

## 一、产品定位与核心价值

### 产品定位
ResearchMate 是一款面向个人投资者、金融分析师和投资顾问的智能投研助手，旨在帮助用户在10分钟内完成对一家公司的基本面、新闻动态和市场情绪的全面分析。

### 核心价值
| 价值维度 | 描述 |
| --- | --- |
| **效率提升** | 自动解析财报，提取关键指标，省去手动翻找时间 |
| **信息整合** | 多平台资讯聚合，统一入口获取财经信息 |
| **数据支撑** | 量化市场情绪，提供客观决策依据 |

### F1 财报智能解读模块

**功能定位**：帮助用户快速获取财报关键数据，生成AI摘要

**核心功能**：
| 功能点 | 描述 | 用户价值 |
| --- | --- | --- |
| PDF上传 | 支持本地PDF文件上传（≤50MB） | 便捷导入财报文档 |
| 智能解析 | 自动识别公司名称、股票代码、报告期等 | 省去手动录入 |
| 指标提取 | 营收、净利润、经营现金流、负债率、毛利率 | 快速掌握财务状况 |
| AI摘要 | 200字中文智能总结 | 快速了解财报重点 |
| 手动补全 | 未识别字段支持手动填写 | 保证数据完整性 |

**用户流程**：
```
用户登录 → 点击上传财报 → 选择PDF文件 → 查看解析进度 → 查看结果页
```

**异常处理**：
- 扫描件PDF：提示"该文件为扫描件，OCR功能将在后续版本支持"
- 解析超时（60s）：提示超时，支持重试
- 未识别字段：显示"未识别"占位，支持手动填写

---

### F2 市场新闻聚合模块

**功能定位**：整合多平台财经资讯，提供统一阅读入口

**核心功能**：
| 功能点 | 描述 | 用户价值 |
| --- | --- | --- |
| 关注列表 | 添加/删除关注公司（支持名称/股票代码） | 个性化资讯定制 |
| 资讯聚合 | 按关注公司/行业展示最新资讯 | 一站式获取信息 |
| 智能筛选 | 按公司/行业筛选资讯 | 精准获取所需内容 |
| 每日推送 | 08:00自动推送新闻小结（可关闭） | 及时掌握市场动态 |

**数据来源**：
1. NewsAPI（免费层）
2. 东方财富数据API
3. 后续可扩展RSS订阅

**资讯卡片信息**：
- 标题
- 来源
- 发布时间
- 情绪分数

---

### F3 情绪指标分析模块

**功能定位**：基于新闻标题进行情感分析，量化市场情绪

**核心功能**：
| 功能点 | 描述 | 用户价值 |
| --- | --- | --- |
| 情感分类 | 单条资讯：正面/中性/负面 | 快速判断资讯倾向 |
| 情绪分数 | 公司维度情感加权平均（-100~+100） | 量化市场情绪 |
| 趋势图表 | 近30天情绪趋势图 | 直观展示情绪变化 |
| 情绪旗帜 | 🟢积极 / ⚪️中性 / 🔴消极 | 快速识别当日情绪 |

**算法方案**：使用SnowNLP或VADER中文适配版进行情感分析
