# ResearchMate 版本计划与路线图

## 文档说明
根据竞品分析结果，本计划重新定位产品为"**个人投资者的财报解读第一站**"，并详细规划了后续开发的时间线和功能里程碑。

---

## 📋 总体路线图概览

| 版本 | 核心主题 | 功能焦点 | 预计周期 | 状态 |
|------|---------|---------|---------|------|
| **v1.0 MVP** | **核心能力完善** | UI优化、深色模式、合规设计 | 1-2周 | ⏳ 进行中 |
| **v1.1 增强版** | **AI交互升级** | 一句话查财报、财报解读Skill | 2-3周 | 📅 计划中 |
| **v1.2 扩展版** | **多Agent能力** | 多风格Agent、工作台 | 2-3周 | 📅 计划中 |
| **v2.0 完整版** | **生态建设** | 微信接入、社区功能 | 3-4周 | 📅 计划中 |

---

## 🚀 v1.0 MVP（当前优先级最高）

### 目标
完善MVP核心功能，重点优化用户体验，符合投研产品设计规范，确保合规。

### 核心功能清单

| 优先级 | 模块 | 功能描述 |
|------|------|---------|
| **P0** | **UI优化** | 便当网格2.0布局，卡片式模块展示 |
| **P0** | **深色模式** | 深色/浅色模式切换，深色优先 |
| **P0** | **进度可视化** | 财报分析进度条展示 |
| **P0** | **结构化展示** | 关键指标卡片/表格展示 |
| **P0** | **合规提示** | AI生成标识 + 风险提示 |
| **P0** | **PDF清理** | 分析后不保存原始PDF文件 |
| **P1** | **首次使用确认** | 服务条款弹窗 |
| **P1** | **响应式优化** | 移动端视图适配 |

### 详细任务分解（待确认）

#### 1. UI/UX 优化（便当网格2.0布局）
- [ ] 首页仪表盘改版，采用卡片式布局
- [ ] 财报解析结果页结构化重新设计
- [ ] 新闻列表卡片式重构
- [ ] 情绪分析页面可视化优化

#### 2. 深色模式支持
- [ ] 设计深色/浅色两套配色方案
- [ ] 实现主题切换功能
- [ ] 保存用户主题偏好
- [ ] 所有页面适配深色模式

#### 3. 进度可视化
- [ ] 财报上传进度条
- [ ] 分析过程状态显示（解析中/提取中/摘要生成中）
- [ ] 预计完成时间提示

#### 4. 合规功能
- [ ] AI生成结果添加标识："AI生成，仅供参考"
- [ ] 添加风险提示："本分析不构成投资建议"
- [ ] PDF分析后自动删除原始文件
- [ ] 首次使用服务条款弹窗确认

## 🎯 v1.1 增强版（接下来开发）

### 目标
从"工具型"向"智能交互型"升级，引入"一句话查财报"和"财报解读Skill"。

### 核心功能清单

| 优先级 | 模块 | 功能描述 |
|------|------|---------|
| **P0** | **自然语言交互** | "一句话查财报" - 支持自然语言查询 |
| **P0** | **财报解读Skill** | 专业方法论封装，标准化分析流程 |
| **P1** | **AI摘要优化** | 提升摘要质量和可读性 |
| **P1** | **历史对比** | 同公司财报历史数据对比 |

#### 自然语言查询示例
```
用户输入："帮我分析一下腾讯2024年Q1财报"
系统响应：自动解析+结构化展示+AI分析

用户输入："这份财报的利润情况如何？"
系统响应：聚焦利润指标的深度解析
```

## 🔮 v1.2 扩展版（未来扩展）

### 目标
引入多Agent分析视角，提供多样化分析能力。

### 核心功能清单

| 优先级 | 模块 | 功能描述 |
|------|------|---------|
| **P0** | **多风格Agent** | 保守/激进/成长/价值四种分析视角 |
| **P0** | **工作台形态** | 多面板拖拽布局 |
| **P1** | **对比分析** | 支持多家公司横向对比 |
| **P1** | **报告导出** | 生成PDF/PPT分析报告 |

### 多Agent分析视角

| Agent类型 | 分析风格 | 适用场景 |
|----------|---------|---------|
| 保守型 | 风险优先，关注安全性 | 价值投资者 |
| 激进型 | 成长优先，关注增长 | 成长型投资者 |
| 平衡型 | 兼顾风险与收益 | 普通投资者 |
| 专业型 | 全维度深度解析 | 专业分析人员 |
