# ResearchMate - 项目功能文档与版本计划

## 📋 项目概述

**ResearchMate** 是一个智能财务分析平台，集成 AI 技术，提供财报解析、新闻聚合、情绪分析等功能。

**当前版本:** v1.0.0 (MVP)  
**部署环境:**
- 前端: Cloudflare Pages (`https://researchmate.pages.dev`)
- 后端: Render (`https://researchmate-aznu.onrender.com`)
- 数据库: PostgreSQL (Render 免费)

## 📊 当前技术栈

### 后端技术
```
框架: FastAPI 0.99.1
数据库: PostgreSQL (SQLAlchemy 1.4.49)
认证: JWT (python-jose 3.3.0)
AI: OpenAI GPT-3.5-turbo
邮件: Resend 0.8.0
部署: Render (Python 3.11)
```

### 前端技术
```
框架: Next.js 14.2.35
UI: Tailwind CSS
状态: React Context
HTTP: Axios + Fetch
部署: Cloudflare Pages
```

### 第三方服务
```
- NewsAPI (新闻聚合)
- OpenAI (AI 分析)
- Resend (邮件发送)
- Render (托管)
- Cloudflare Pages (CDN)
```

## 🔧 配置指南

### 必需的环境变量

**Render 后端:**
```bash
# 数据库
DATABASE_URL=postgresql://user:pass@host:5432/dbname

# 安全
SECRET_KEY=your-super-secret-key-here

# AI 服务
OPENAI_API_KEY=sk-xxxxxxxxxxxxx

# 邮件服务 (新增)
RESEND_API_KEY=re_xxxxxxxxxxxx
RESEND_FROM_EMAIL=noreply@researchmate.dev

# 新闻服务
NEWS_API_KEY=your-news-api-key
```

**Cloudflare Pages 前端:**
```bash
NEXT_PUBLIC_API_URL=https://researchmate-aznu.onrender.com
```

## 🐛 已知问题 & 限制

### v1.0.0 已知问题
1. ⚠️ **AI 查询降级模式**
   - 问题: OpenAI proxies 参数兼容性问题
   - 影响: 使用本地 fallback 回答
   - 修复计划: v1.1.0

2. ⚠️ **邮件发送依赖配置**
   - 问题: 需要 RESEND_API_KEY 才能发送真实邮件
   - 影响: 未配置时显示"邮件服务暂时不可用"
   - 解决: 按照[Resend 注册指南](https://resend.com)获取 API Key
